# 🏡 PLOTROL – Property Inspection & Sale Verification App  

### 📱 *A Flutter-based mobile application for property inspection and verification *  

---

## 🚀 Overview  

**PLOTROL** is a mobile application built using **Flutter**, designed to streamline the **property inspection and sale verification** process.  

It enables **plot owners** to raise inspection requests for their property, which are then managed by **admins** and **help desk inspectors** through a structured workflow.  
The app ensures transparency and accuracy in property assessments with integrated location capture, file uploads, and secure authentication.

---

## 🔧 Key Features  

- 🔐 **Authentication** – Secure login and role-based access for users, admins, and help desk members.  
- 🧾 **Service Request Workflow** – End-to-end process from request creation to inspection approval.  
- 📍 **Location Capture** – Automatically captures the property’s coordinates during inspection.  
- 📁 **File Upload** – Attach inspection images, reports, or supporting documents.  
- 🧰 **Admin Dashboard** – Manage and assign inspection requests efficiently.  
- 🗒️ **Checklist & Remarks** – Structured inspection criteria and feedback submission.  
- ⚙️ **Backend Integration** – Real-time service request and data synchronization.  

---

## 🧭 Workflow Summary  

1. **Property Owner** submits an inspection request for their plot.  
2. **Admin** reviews the request and assigns a **Help Desk User**.  
3. **Help Desk User** visits the location, inspects the property, fills the checklist, adds remarks, and uploads photos/files.
